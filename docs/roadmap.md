# Roadmap

## Phase 1

Build the architecture skeleton and minimal working demos:
- core constants, units, species
- particle cloud state
- parameterized positron and surrogate antiproton sources
- uniform and idealized solenoid fields
- relativistic Boris solver
- aperture and momentum-window acceptance
- pipeline loss accounting
- accepted-yield diagnostics
- placeholder adapters only

## Later Phases

Potential future work:
- richer validation cases
- optional file I/O extras
- table-based source terms
- externally calibrated source terms
- adapter-backed comparison workflows
- visualization and sweep utilities
